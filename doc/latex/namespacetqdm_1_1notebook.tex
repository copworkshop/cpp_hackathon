\doxysubsection{Detailed Description}
\begin{DoxyVerb}IPython/Jupyter Notebook progressbar decorator for iterators.
Includes a default `range` iterator printing to `stderr`.

Usage:
>>> from tqdm.notebook import trange, tqdm
>>> for i in trange(10):
...     ...
\end{DoxyVerb}
