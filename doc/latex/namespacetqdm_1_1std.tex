\doxysubsection{Detailed Description}
\begin{DoxyVerb}Customisable progressbar decorator for iterators.
Includes a default `range` iterator printing to `stderr`.

Usage:
>>> from tqdm import trange, tqdm
>>> for i in trange(10):
...     ...
\end{DoxyVerb}
