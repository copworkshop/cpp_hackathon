\doxysubsection{Detailed Description}
\begin{DoxyVerb}An output path that may be empty.

- ``None``: the option is not set
- ``OutputOrDefault(None)``: fall back to some default value
- ``OutputOrDefault(path)``: use that path
\end{DoxyVerb}
