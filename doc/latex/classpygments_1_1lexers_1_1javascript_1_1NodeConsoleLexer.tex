\doxysubsection{Detailed Description}
\begin{DoxyVerb}For parsing within an interactive Node.js REPL, such as:

.. sourcecode:: nodejsrepl

    > let a = 3
    undefined
    > a
    3
    > let b = '4'
    undefined
    > b
    '4'
    > b == a
    false

.. versionadded: 2.10
\end{DoxyVerb}
